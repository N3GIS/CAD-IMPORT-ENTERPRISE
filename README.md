# CAD IMPORT ENTERPRISE
# SHORT DESCRIPTION:
Efficient Unity3D 3D CAD format loading plugin, supporting CAD formats such as Catia, SolidWorks, Inventor, Rhino, JT, and PARASOLID. It also supports PIM, importing at different levels of precision, and offline use.

# Features:
1. Support STEP (AP203, AP214, AP242) 3D file format (.stp, .step)
2. Support the import of PMI attributes of STEP （AP203、AP214）files
3. Support the import of PMI attributes of CATIA （V5-6R2023、V5-6R2019、V5-6R2014、V5 R21 ） files
4. Support the import of PMI attributes of NX（NX 2027、NX 1980、NX 9、NX 8 ） files
5. Support the import of PMI attributes of Autodesk Inventor（Inventor 2024、Inventor 2021） files
6. Support the import of PMI attributes of SolidWorks（SolidWorks MBD 2018、SolidWorks 2015、SolidWorks 2012 ） files
7. Support for IGES 3D file format (.Igs,.Iges)
8. Support for Catia V4 3D file format (.model,.dlv,.dlv3,.exp,.session)
9. Support for Catia V5 3D file format (.CATPart,.CATProduct)
10. Support for Catia V6/3DEXPERIENCE 3D file format (.3dxml)
11. Support for PARASOLID 3D file format (.x_t,.x_b,.xmt_txt,.xmt_bin)
12. Support for SolidWorks 3D file format (.sldprt,.sldasm)
13. Support for Solid Edge 3D file format (.par,.psm,.asm,.sldasm)
14. Support for ProE/Creo Parametric 3D file format (.par,.xpr,.asm,.xas)
15. Support for DWG/DXF 3D file format (.dwg,.dxf)
16. Support for Inventor 3D file format (.ipt,.iam)
17. Support for 3DVIA/SW/Catia Composer file format (.smg,.smggeom)
18. Support for Rhino 3D file format (.3dm)
19. Support for STL 3D file format (.stl)
20. Support for UG NX 3D file format (.prt)
21. Support for JT file format (.jt)
22. Support for glTF file format (.gltf,.glb)
23. Support for IFC file format (.ifc,. ifcxml,.ifczip)
24. Support for Revit file format (.rvt,.rfa)
25. Support for Navisworks file format (.nwd)
26. Support for FBX file format (.fbx,.dae,.obj)
27. Support for Unity editor import
28. Support for runtime import
29. Support model material import
